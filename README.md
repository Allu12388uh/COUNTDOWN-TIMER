# COUNTDOWN-TIMER:

This is a simple Countdown Timer project that counts down the time until the New Year.
The timer is built using HTML, CSS, and JavaScript. It dynamically updates the days, hours, minutes, and seconds remaining until the start of the new year.


Features:

Countdown to New Year: The timer counts down to January 1 of the next year.

Responsive Design: The layout adjusts for different screen sizes (mobile, tablet, desktop).

Styled with CSS: Modern styling with Google Fonts and background image.

Real-time updates: The countdown is updated every second.


Technologies Used:

HTML5: Used for the structure of the web page.

CSS3: Provides the styling for the timer and page layout, including a responsive design.

JavaScript: Powers the countdown logic, updating the displayed time in real-time.

Google Fonts: Poppins and Inspiration fonts used for styling text.


File Structure:

index.html: Main HTML file for the countdown timer layout.

css/style.css: CSS file containing styles for the timer and page.

js/script.js: JavaScript file responsible for the countdown logic.

background.jpg: Image used as the background for the page.
